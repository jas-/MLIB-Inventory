Properties

Name Type Default
value Number 0

Gets or sets the progress bar's value The value should be set between min(default value: 0) and max(default value: 100).

Code examples

Initialize a jqxProgressBar with the value property specified. 

$('#jqxProgressBar').jqxProgressBar({ width: 200, height: 25, value: 100});
max Number 100

Sets the progress max value.

Code examples

Initialize a jqxProgressBar with the max property specified. 

$('#jqxProgressBar').jqxProgressBar({ width: 200, height: 25, max: 150 });
min Number 0

Sets the progress min value.

Code examples

Initialize a jqxProgressBar with the min property specified. 

$('#jqxProgressBar').jqxProgressBar({ width: 200, height: 25, min: 10 });
orientation String 'horizontal'

Gets or sets the orientation.

Code examples

Initialize a jqxProgressBar with the orientation property specified. 

$('#jqxProgressBar').jqxProgressBar({orientation: "vertical"  });
width Number/String null

Gets or sets the progress bar width.

Code examples

Initialize a jqxProgressBar with the width property specified. 

$('#jqxProgressBar').jqxProgressBar({width:"250px"  });
height Number/String null

Gets or sets the progress bar height.

Code examples

Initialize a jqxProgressBar with the height property specified. 

$('#jqxProgressBar').jqxProgressBar({height:"100px"  });
rtl Boolean false
Gets or sets a value indicating whether widget's elements are aligned to support locales using right-to-left fonts.

Code example

Set the "rtl" property. 

$('#widget').jqxProgressBar({rtl : true});

Get the "rtl" property. 

var rtl = $('#widget').jqxProgressBar('rtl');
theme String ''

Sets the widget's theme.

jQWidgets uses a pair of css files - jqx.base.css and jqx.[theme name].css. The base stylesheet creates the styles related to the widget's layout like margin, padding, border-width, position. The second css file applies the widget's colors and backgrounds. The jqx.base.css should be included before the second CSS file. In order to set a theme, you need to do the following:
  • Include the theme's CSS file after jqx.base.css.
    The following code example adds the 'energyblue' theme. 
    

    <link rel="stylesheet" href="../../jqwidgets/styles/jqx.base.css" type="text/css" />
<link rel="stylesheet" href="../../jqwidgets/styles/jqx.energyblue.css" type="text/css" />
  • Set the widget's theme property to 'energyblue' when you initialize it.
showText Boolean false

Gets or sets the visibility of the progress bar's percentage's text.

Code examples

Initialize a jqxProgressBar with the showText property specified. 

$('#jqxProgressBar').jqxProgressBar({showText:true});
layout String "normal"

Sets or gets the jqxProgressBar's layout. Possible values: "normal" and "reverse". When the layout is set to "reverse", the slider is filled from right-to-left(horizontal progressbar) and from top-to-bottom(vertical progressbar)

Code examples

Initialize a jqxProgressBar with the layout property specified. 

$('#jqxProgressBar').jqxProgressBar({ layout: "reverse" });
animationDuration Number 300

Determines the duration of the progressbar's animation.

Code examples

Initialize a jqxProgressBar with the animationDuration property specified. 

$('#jqxProgressBar').jqxProgressBar({animationDuration:0});
disabled Boolean false

Determines whether the progress bar is disabled.

Code examples

Initialize a jqxProgressBar with the disabled property specified. 

$('#jqxProgressBar').jqxProgressBar({disabled:true});

Events
valuechanged Event

This event is triggered when the value is changed.

Code examples

Attach a handler to the valuechanged event by type: jqxProgressBar. 

$('#jqxProgressBar').on('valuechanged', function (event) { var value = event.currentValue; });
invalidvalue Event

This event is triggered when the user enters an invalid value( value which is not Number or is out of the min - max range. )

Code examples

Attach a handler to the invalidvalue event by type: jqxProgressBar. 

$('#jqxProgressBar').on('invalidvalue', function (event) { // Some code here. });
complete Event

This event is triggered when the value is equal to the max. value.

Code examples

Attach a handler to the complete event by type: jqxProgressBar. 

$('#jqxProgressBar').on('complete', function (event) { // Some code here. });

Methods
actualValue Method

Sets the progress bar's value

Code examples

Invoke the actualValue method. 

//@param Number
$('#jqxProgressBar').jqxProgressBar('actualValue', 50);
destroy Method

Destroys the widget.

Code examples

Invoke the "destroy" method. 

$('#widget').jqxProgressBar('destroy');
